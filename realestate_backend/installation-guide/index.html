Real Estate CRM

A Simple Real Estate CRM

Real Estate CRM is a simple CakePHP application to track all the activities of sales and purchase of property for clients. It offers basic but extremly essential features for real estate agents/companies.

Installation Guide

Installation Requirement

  • PHP 5.4 or Higher
  • MYSQL 5.0 or Higher
  • PDO Extension
  • GD Library

Installation Procedure

  • Unzip the package realestate-crm
  • Upload the package-xx.zip files to the directory of your web server.
  • Unzip the file and you are good to go.
  • Now access the directory where you have uploaded the package. If you uploaded in your root directory the url will be http://www.yourdomain.com/ or if you chosen to upload it on sub directory it should be http://www.yourdomain.com/directoryyouchosen/
  • Now create an user/password and database on MYSQL (For Cpanel Users Refer : Click here to view Cpanel Database Guide ). For rest other you can contact hosting provider if you don't know what this document talking about.

Installation Wizard

Step One

You will be able to view the installation screen with first step to check the compatibility environment of the server. It also checks the required permissions and extension to complete the installation.

Step Two

You will be able to view the installation screen with first step to accept the agreement. Read the agreement carefully and check I Accept to Submit and proceed to next step

Step Three

Host Name: Generally Localhost

Database Name: Name of the Database created for Real Estate CRM

DB User: User name associated with the above database for Real Estate CRM

DB Password : Password for above user name

Database Type : Database Engine Type



Step Four

Fill the essential branding info like Site Name, Currency, Organization Email, Time zone etc.

Yepee the installation is complete. You may login the admin now.
